Extend
======

**Alias:** ``E X``

Extends objects to meet a selected boundary edge.

.. image:: ../../icons/extend.svg
   :width: 48
   :alt: Extend icon

----

Description
-----------

The Extend command lengthens objects so that their endpoints meet a specified boundary edge. First select the boundary edges, then pick the objects to extend. The object is extended from the end nearest to the point you click.

Workflow
--------

1. Type ``E X`` and press ``Space`` or ``Enter``.
2. **Select boundary edges:** Click the objects to extend to, then press ``Enter``.
3. **Select objects to extend:** Click the end of each object you want to lengthen toward the boundary.
4. Continue clicking objects to extend, then press ``Enter`` or ``Escape`` to end.

Tips
----

- Click near the end of the object you want extended; the nearest endpoint is extended toward the boundary.
- If the object cannot be extended to reach the boundary (e.g. parallel lines), it will not be modified.
- Use :doc:`trim` to shorten objects at a boundary instead of lengthening them.
