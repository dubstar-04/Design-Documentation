Zoom
====

**Alias:** ``Z``

Changes the magnification of the active view.

----

Description
-----------

The Zoom command adjusts how much of the drawing is visible on screen, without moving or modifying any objects. You can zoom in to work on fine detail or zoom out to see the whole drawing.

Workflow
--------

1. Type ``Z`` and press ``Space`` or ``Enter``.
2. **Specify zoom option** — pick a point to zoom into, or use the scroll wheel at any time.

You can also zoom at any time without running the command:

- **Scroll wheel up** — zoom in centred on the cursor position.
- **Scroll wheel down** — zoom out centred on the cursor position.

Tips
----

- Scrolling with the mouse wheel is the quickest way to zoom during drawing without interrupting your current command.
- Use :doc:`pan` to move the view after zooming in.
- Zoom does not affect the actual size or position of objects in the drawing.
